# Project Roadmap

- M3-C3: complete the three-ToF replay log pipeline.
- M3-C3.1: align real protocol, range, FOV, timestamp, and freshness semantics.
- M3-C4: connect real-format three-ToF replay into the fake autonomous pipeline.
- M3-D0: build closed-loop 2D simulation.
- M3-D1: wrap legacy Localizer, TofFilter, and ChunkedGrid as a lightweight sparse ToF SLAM backend.
- M3-D2: add sparse scan matching, pose correction, and keyframes.
- M3-E: add Frontier and A* exploration.
- M3-F: complete simulation acceptance.

M3-C3.1 is not real hardware enablement, production SLAM, real map writing, pose writeback, or autonomous exploration.
